## Welcome to the DynamoPDF API

The DynamoPDF API is a comprehensive PDF generation service that allows you to create, manage, and generate PDF documents from templates with dynamic data binding. Our API provides powerful template management with secure file storage.

## Quick Start

### 1. Get Your API Key

First, create an API key in the [Developer Portal](https://app.dynamopdf.com/developer):

### 2. Create a Template

Use DynamoPDF's drag and drop editor to create a pdf template.

### 3. Generate a PDF

```bash
curl -X POST "https://api.dynamopdf.com/api/generate-pdf" \
  -H "X-API-Key: dk_live_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "templateId": "template_123456789",
    "data": {
      "invoiceNumber": "INV-2024-001",
      "customerName": "John Doe",
      "amount": 1250.00
    },
    "filename": "invoice_2024_001.pdf"
  }'
```

## Base URL

All API requests should be made to:

```
https://api.dynamopdf.com
```

## Rate Limiting

API endpoints have different rate limits based on their computational requirements:

| Endpoint Type | Rate Limit | Time Window |
|---------------|------------|-------------|
| Template Management | 100 requests | 1 minute |
| PDF Generation | 10 requests | 1 minute |
